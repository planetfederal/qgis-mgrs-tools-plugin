about=<p>The Military Grid Reference System (MGRS) was created as a geocoordinate standard used by NATO militaries for locating points on the earth. MGRS coordinates are a single string composed of a pattern of letters and numbers representing a grid square.</p><p>The MGRS Map Tool displays MGRS coordinates in the QGIS status bar (bottom left) as you pan and zoom around the map. The Zoom to MGRS coordinate tool allow you to zoom the map canvas to a given MGRS coordinate.</p>
